# Sudoku Solver
A program to solve sudoku puzzles.

## Solver Usage 
```
Sudoku Solver (GUI)
usage: solver.py [-h] [-p] [-d] [-f FILE]

options:
  -h, --help            show this help message and exit
  -p, --progress        Display progress in terminal
  -d, --debug           Display debug information
  -f FILE, --file FILE  File to load board from
```

```
Sudoku Solver (In terminal)
usage: display.py [-h] [-p] [-d] [-f FILE]

options:
  -h, --help            show this help message and exit
  -p, --progress        Display progress in terminal
  -d, --debug           Display debug information
  -f FILE, --file FILE  Path to the grid file
```
